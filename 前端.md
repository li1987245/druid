### npm
- 查看全局安装模块
```markdown
npm list -g --depth 0
```
- 查看npm默认设置
```markdown
npm config ls -l
修改全局安装路径
npm config set prefix "D:\Program Files\npm_global_modules\node_modules"
```
- 卸载模块
```markdown
npm uninstall -g <package>
```
